Welcome to xCover Python SDK’s documentation!¶

To use this library

from xcovlib.client import Client

key = 'KEY'
secret = 'SECRET'
host = '127.0.0.1'
client = Client(key, secret, host)

Now you can continue to use the methods below normally

Client¶

Base Class which lets you access the xCover-Core Client and make requests to the service.

>>> client = Client(host=host, key=key, secret=secret)
